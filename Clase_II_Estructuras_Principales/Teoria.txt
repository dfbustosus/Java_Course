# Tipos de datos
Son 8 en total
1. Enteros
1.1 Int: 4 bytes desde -2147483648 hasta 2147483647
1.2 Short: 2 bytes desde -32768 hasta 32767
1.3 Long: 8 bytes se ponen con sufijo L. e.g 50000000L
1.4 Byte: 1 byte desde -128 hasta 127

2. Floats
2.1 Float: 4 bytes con entre 6 a 7 cifras decimales, se crean con sufijo F
2.2 Double: 8 bytes con ~15 cifras decimales

3. Char
Representan caracteres unicode (esto es un tema denso pero no me voy a meter en eso para el curso)

4. Boolean
Son true or false 

Otros tipos de datos como fechas, strings no son DATOS PRIMITIVOS
SON OBJETOS. Importante recordarlo

Consejo: La mejor forma de aprender esto es con la práctica, no lo memorices

# Variables
¿Qué es? Espacio de memoria del ordenador 
¿Para que la usamos? Se usan para guardar data de forma temporal
¿Como se crea? Se crea especificando el tipo de dato + nombre Variable e.g int salario;
¿Como iniciar variables? Para inicializar la variables nombre_variable=valor. e.g salario= 10;

# Importante
Ocupa las variables de forma eficiente para no consumir demasiada RAM
